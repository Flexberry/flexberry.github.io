    EnumNumberTransform Class

    Extends EnumBaseTransform
    Defined in: ember-flexberry/addon/transforms/enum-number.js:4
    Module: ember-flexberry

    Base transform class that implements an enumeration type for numbers. During serialization and deserialization it checks if value is not outside of accessible values and converts to number or back.
