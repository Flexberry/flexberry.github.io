    FlexberrySimpledatetime Class

    Extends FlexberryBaseComponent
    Defined in: ember-flexberry/addon/components/flexberry-simpledatetime.js:8
    Module: ember-flexberry

    Wrapper for input[type='date/datetime/datetime-local'] component.

    Please keep in mind that these input types are not supported in all browsers: http://caniuse.com/#feat=input-datetime.
