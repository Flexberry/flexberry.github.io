    NewPlatformFlexberryGISMapLayerSettingRouteMixin Class

    Uses
    • <a href="http://emberjs.com/api/classes/Ember.Mixin.html">Ember.Mixin</a>
    Defined in: ember-flexberry-gis/addon/mixins/new-platform-flexberry-g-i-s-map-layer-setting-route.js:7
    Module: ember-flexberry-gis

    Mixin for DS.Route to support work of component NewPlatformFlexberryGISMapLayerSettingComponent.

    Now it supports following layer types:

    • 'tile' (name of model and template new-platform-flexberry-g-i-s-map-layer-tile');
    • 'wms' (name of model and template new-platform-flexberry-g-i-s-map-layer-wms').

    In order to add support of new layer it is necessary to:

    • add corresponding model to "models" folder;
    • add corresponding template to "templates" folder;
    • add such variant to actions.renderMainTemplate.
