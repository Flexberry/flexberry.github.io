    ExportMapCommandComponent Class

    Uses
    • <a href="https://github.com/ciena-blueplanet/ember-block-slots#usage">SlotsMixin</a>
    Extends Ember.Component
    Defined in: ember-flexberry-gis/addon/components/map-commands/export.js:30
    Module: ember-flexberry-gis

    Flexberry export map-command component. Component must be used in combination with {{#crossLink "FlexberryMaptoolbarComponent"}}flexberry-maptoolbar component{{/crossLink}} as a wrapper.

    Usage: templates/my-map-form.hbs

    {{#flexberry-maptoolbar leafletMap=leafletMap as |maptoolbar|}}
  {{map-commands/export execute=(action "onMapCommandExecute" target=maptoolbar)}}
{{/flexberry-maptoolbar}}
