    FileTransform Class

    Extends DS.StringTransform
    Defined in: ember-flexberry/addon/transforms/file.js:7
    Module: ember-flexberry

    This transformation is necessary in order to detach file attributes from another string attributes on model level. It extends string transformation from ember data (https://github.com/emberjs/data/blob/v2.3.0/addon/-private/transforms/string.js), without any changes.
