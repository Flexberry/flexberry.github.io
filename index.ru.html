Инструменты проектирования и программирования

Flexberry Designer

Инструмент проектирования и генерации кода (используется нотация UML). Позволяет быстро создать объектную модель структуры данных и форм приложения, сгенерировать работающий прототип.

Фреймворки

Flexberry Ember

Фреймворк для создания SPA-приложений.

Flexberry ASP.NET

Фреймворк для создания ASP.NET WebForms приложений.

Flexberry Winforms

Фреймворк для создания Winforms-приложений.

Функциональные подсистемы

Flexberry Foundation

Учетная часть бизнес-приложений.

Flexberry GIS

Подсистема для ГИС-приложений.

Flexberry Service Bus

Подсистема интеграции приложений.

Flexberry Analytics

Подсистема отчётности и бизнес-аналитики приложений.

Flexberry BPM

Подсистема для процессной части приложений.

Flexberry Portal

Подсистема для создания корпоративного портала.

Технологические компоненты

Flexberry ORM

ORM с расширенной поддержкой проекций и различных типов СУБД.

Flexberry Security

Компоненты для разграничения прав пользователей.

Flexberry Audit

Компоненты для аудита изменения данных.
